So, our today's topic is lists in HTML.


Firstly, we will begin with lists.


There are two types of lists "ul" and "ol".

Here, "ul" refers to Unordered List and "ol" refers to Ordered List.

Let's start with Unordered List.


Explaination:- There are three types of Unordered List:- disc, square and circle.


Disc:- It's a circle which is fully filled from inside like bullets.


Square:- It's a sqaure which is also filled from inside.


Circle:- It's a circle which is not filled (empty) from inside.


We can access these three types of Unordered List by typing the command ('ul type=" you will type here any three of them" ')


You can make your lists inside "ul" tag by typing the command (li)


Now, I will show you example of Unordered List with these three types.

So, this was our Unordered List.

Let's move onto Ordered List.


Explaination:- As it's evident by it's name that it is displayed in te correct order or sequence like => 1,2,3... and a,b,c,... and A,B,C... etc.


It contains two elements in it which are "type", "start" and "reversed"


You can access these command by typing 'ol type="" start="" reversed="" '
